# ifood-spent

ifood-spent is a script that show in your terminal how much spent in ifood

## Installation

Yarn
```bash
yarn install
```

Npm
```bash
npm install
```

## Config

Set your ACCESS_TOKEN of ifood in venv `TOKEN` in `.env`

## Use

After set your access_token, just run:

```bash
node index.js
```
